# 静态资源说明

请将以下图片文件放置到此目录：

## 必需文件

### 1. logo.jpg
- 用途：顶部栏 Logo 图标
- 尺寸建议：200x200 像素
- 格式：JPG 或 PNG
- 说明：小程序品牌 Logo，建议使用暖色调

### 2. share-cover.png
- 用途：分享卡片封面图
- 尺寸建议：500x400 像素（5:4 比例）
- 格式：PNG
- 说明：微信分享时显示的封面图，建议包含品牌信息和核心功能描述

## 图片设计建议

### 色彩规范
- 主色调：#FF6B35（橙红色）
- 辅助色：#F7931E（暖橙色）
- 背景色：#FFF8F0（米白色）

### Logo 设计要点
- 简洁明了，突出"午餐"主题
- 可考虑使用饭盒、筷子等元素
- 避免过于复杂的细节

### 分享封面设计要点
- 标题：午餐吃啥外卖
- 副标题：10秒生成3套午餐方案
- 包含品牌 Logo
- 背景使用渐变色更有质感
